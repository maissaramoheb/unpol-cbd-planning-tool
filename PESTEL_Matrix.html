UNPOL CBD Integrated Planning Tool

This offline educational and planning tool links three modules: UNPOL Situational Analysis, UNPOL Stakeholder Analysis & Mapping, and the UNPOL CBD Matrix. It is designed to support context understanding, actor analysis, and targeted CBD prioritization.

Module 1

Use UNPOL-adapted PESTEL to identify environmental pressures affecting host-state policing.

Module 2

Assess which actors enable, block, or shape the feasibility of police support and reform.

Module 3

Translate context and actor analysis into CBD action across key areas, dimensions, and sectors.

Start with context, then move to actors, then use the CBD matrix as the implementation logic.
